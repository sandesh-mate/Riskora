# Riskora
Portfolio Risk Intelligence System
Riskora is a data analytics project focused on analyzing financial market risk, measuring portfolio volatility, and forecasting future performance using statistical and simulation techniques.
The system helps in making data-driven investment decisions by providing insights into risk, returns, and diversification.
> Key Features
  Real-time stock data collection using yfinance
  Daily log returns and portfolio performance analysis
  Value at Risk (VaR) at 95% and 99% confidence levels
  Monte Carlo Simulation for future portfolio forecasting
  Correlation heatmap for diversification insights
  Rolling volatility analysis
  Interactive dashboard using Power BI
> Technologies Used
   Python
   Pandas
   NumPy
   Matplotlib
   Seaborn
   Power BI
   yfinance
> Project Workflow
  Data Collection → Data Cleaning → Log Return Calculation → Covariance Matrix → Monte Carlo Simulation → VaR Calculation → Visualization
